# PDX1 Variant Analysis in Type 2 Diabetes

## Research Title
Detection of the PDX1 E224K Variant in Patients with Type 2 Diabetes Mellitus

## Overview
This repository contains my undergraduate molecular genetics research investigating the PDX1 E224K variant associated with Type 2 Diabetes Mellitus.

## Techniques Used
- DNA extraction
- Polymerase Chain Reaction (PCR)
- Agarose gel electrophoresis
- Sanger sequencing
- DNA sequence analysis using BioEdit
- Statistical analysis using SPSS

## Software
- BioEdit
- SPSS
